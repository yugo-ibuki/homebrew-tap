# Yugo-ibuki Tap

## How do I install these formulae?

`brew install yugo-ibuki/tap/<formula>`

Or `brew tap yugo-ibuki/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "yugo-ibuki/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
